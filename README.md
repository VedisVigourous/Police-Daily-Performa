# Police-Daily-Performa
### Access Here: [Daily Performa](https://vedisvigourous.github.io/Police-Daily-Performa/DailyPerforma/)

---
## Overview

Simple web app for Delhi Police staff to fill the daily Late Arrival of Local Police Performa.

## Features

- Clean table-style form for entering report details
- Daily note section at the end of the form
- Image download and print / PDF export support
- Layout designed for quick sharing and record keeping

## How to Use

1. Open the live demo in a browser.
2. Fill in the table fields and daily note.
3. Download the report as an image or use print / PDF export.
