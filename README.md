# Police-Daily-Performa

Simple web form for filling the daily Late Arrival of Local Police Performa used by Delhi Police staff.

## Purpose

This page is designed for quick daily data entry, clean screenshots, and easy image or PDF export.

## Features

- Editable table-style form with all report fields
- Daily note section at the end of the form
- Image download and print / PDF export support
- Clean layout for sharing and record keeping

## Usage

Open `index.html` in a browser, fill in the fields, add the daily note, then use the image download or print button as needed.
